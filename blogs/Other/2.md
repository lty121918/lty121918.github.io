### 安装步骤[vuepress博客主题—vuepress-theme-reco - 知乎 (zhihu.com)](https://zhuanlan.zhihu.com/p/111749216)

```c
# init
npm install @vuepress-reco/theme-cli -g
theme-cli init my-blog

# install
cd my-blog
npm install

# run
npm run dev

# build
npm run build
```
